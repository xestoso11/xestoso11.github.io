¿como harias un juego que genere un numero aleatorio y que al introducirlo tu te diga si acertaste o no?

Índice

solución

Esta es una de las formas para resolver el suiguiente ejercicio


print("este es un programa que genera un numero aleatorio haber si aciertas suerte !!")

import random

while True:

  nun=float(input("elige tu numero"))

  al= random.randint (1,10)

  if nun==al:

    print ("acertaste")

    break

  else:

    print ("nunca acertaras jajajaja perdiste")

CSS (en inglés Cascading Style Sheets) es lo que se denomina lenguaje de hojas de estilo en cascada y se usa para estilizar elementos escritos en un lenguaje de marcado como HTML.
